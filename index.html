What is Python?

Python is a high-level programming language often used to build websites and software, automate tasks, and conduct data analysis. Python is also a general-purpose language, meaning it can be used to create a variety of different programs and isn't specialized for any specific problem.

Python is popular for a number of reasons. These are some of which that makes Python so versatile and easy to use for coders.

  • It has simple syntax that mimics natural language.
  • It's versatile. Python can be used for many different tasks.
  • It's beginner friendly, making it popular for entry level coders.
  • It's open source, making it free to use and distribute.
  • It's archive of modules and libraries is vast and growing.
  • It has a large and growing community.
Why use Python?

Python is not only used in fields like web-testing, data extraction, artificial intelligence and data science researches, database management, and cybersecurity, it is also used by many famous companies such as Google, Spotify, Youtube, Facebook, and even NASA.

Variables

A variable is like a box in the computer's memory where you can store a single value. You can name a variable anything as long as it obeys the following three rules:

  1. It can be only one word.
  2. It can use only letters, numbers, and the underscore(_) chracter.
  3. It can't begine with a number.

Example of valid variables:

bacon
baconEggs
bacon_eggs
Declaring Variables

Python has no command for declaring variables. A variable is created the moment you first assign a value to it.

You can declare a variable by simply assigning a value to it.

x = 30
y = 21
z = 102
Data Types

Data types are the classification or categorization of data items. It represents the kind of value that tells what operations can be performed on a particular data. Since everything is an object in Python programming, data types are actually classses and variables are instances (object) of these classes.

In Python there are 6 data types:

  • Numeric. Integer, Complex Number or Float.
  • Dictionary. Collection of Key and Value pairs.
  • Boolean. True and False.
  • Set. Unordered collection of items.
  • Sequence Type. String, List, or Tuple.
if...else Statement

The if...elese statement evaluates the test expression and will execute the set of code inside the "if" block only when the test condition is true. If the condition is false, the set of code will not be executed.

if (condition == true):
Block of code
else:
Block of code

Here if the condition is true, the statement inside the "if" block will be executed. If false, the condition inside the "else" block will be executed.

elif Statement

In Python we have one more conditional statement called "elif" statements. "elif" statement is used to check multiple conditions only if the given condition is false. It is similar to an "if-else" statement, the only difference is that in "else" we will not check the condition but in "elif" we will check the condition.

if (condition1 == true):
Block of code
elif (condition2 == true):
Block of code
else:
Block of code

Here the block of code in "elif" will only be executed if the condition in "if" is false. If both "if" and "elif" are false, then the block of code in "else" will be executed.

Example using elif Statement:

num = 10
if (num == 0):
print("Number is zero")
elif (num > 5):
print("Number is greater than 5")
else:
print("Number is lesser than 5")

For a declared variable 'num' with a value of '10', the output of this code will be 'Number is greater than 5'. In this example our declared variable has a value not equal to 0, therefore the outcome of "if" will be false.

As the outcome of our "elif" statement is true, because the value of our variable is greater than 5, the statement inside "elif" will be executed. You can experiment with this code more and change the output by changing the value of 'num'.

while Loop Statement

Python while Loop is used to execute a block of statements repeatedly until a given condition is satisfied. And when the condition becomes false, the line immediately after the loop in the program is executed.

while (condition == true):
block of code

Example using while Loop Statement:

count = 0
while (count < 10):
count = count + 1
print(count)

The output of this program will count from 1 to 10. For a declared variable 'count' with a value of 0, the condition for the while loop is not satisfied as 'count' is less than 10.

While 'count' is still less than 10, the block of code within the while loop will increment 'count' by 1 and output the value of count. This will keep looping until 'count' reaches a value of 10 and the while loop condition is satisfied.
